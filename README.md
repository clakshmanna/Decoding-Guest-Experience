# Decoding Guest Experience

A Django-based web application that performs sentiment analysis on customer reviews using Machine Learning and Natural Language Processing (NLP). The application predicts whether a customer review expresses a **Positive** or **Negative** sentiment, helping businesses understand customer feedback and improve user experience.

---

## 📌 Features

- User-friendly web interface built with Django
- Customer review sentiment prediction
- Machine Learning-based classification
- Natural Language Processing (NLP) techniques
- Displays prediction results instantly
- Easy-to-use interface

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/clakshmanna/Decoding-Guest-Experience.git
```

### Navigate to the project

```bash
cd Decoding-Guest-Experience
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run migrations

```bash
python manage.py migrate
```

### Start the development server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
